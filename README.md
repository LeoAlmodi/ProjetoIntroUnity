# Projeto Intro Unity

Jogo de sobrevivência top-down onde você deve coletar 4 kits médicos antes que o tempo acabe, desviando dos zumbis.

## Como Jogar
- WASD ou setas para mover
- Colete os 4 kits médicos para vencer
- Evite os zumbis
- Pressione R para reiniciar
